# Pathway Foundation

Production-grade Next.js + Vercel + Postgres starter, opinionated for Claude Code agentic development.

## Quick start

1. `pnpm install`
2. `pnpm setup` (interactive — fills `.env.local`)
3. `pnpm db:push`
4. `pnpm dev`
